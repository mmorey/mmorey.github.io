With the recent release of Alfred V2 came Workflows which allow you to add extra functionality to the already powerful keyboard tool. There are already some really great Workflows available:

  • Weather by David Ferguson
  • Emoji Codes by Carlos Galdino
  • Recent Downloads by Dajun Duan
  • Spotify by Jeff Johns

With anticipation of WWDC 2013 I created a rather simple Workflow called WWDC Checker to check the WWDC page. I'm gladly accepting pull requests. The Workflow also supports the Alleyoop updater Workflow for simpler updating.
